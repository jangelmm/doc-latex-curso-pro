\subsection{Listas}

Para definir listas usamos

\begin{itemize}
    \item Linea 1
    \item[!] Linea 2
    \item[*] Linea 3
\end{itemize}

Enumerar cosas

\begin{enumerate}[a)]
    \item Linea 1
    \item Linea 2
    \item Linea 3
\end{enumerate}

Combinar

\begin{enumerate}[a.]
    \item Linea 1
    \begin{itemize}
        \item Linea 1.1
        \item Linea 1.2
        \item Linea 1.3
    \end{itemize}
    \item Linea 2
\end{enumerate}
